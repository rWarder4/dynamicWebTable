Dynamická tabulka

Informace

  • Autor: Jiří Richter, xricht19
  • Název: Dynamická tabulka
  • Skript vznikl jako školní projekt pro předmět WAP - Internetové aplikace.

Použití

Skript dynTable.js přidá libovolné tabulce definované v jazyce HTML možnosti vyhledávání a filtrování. Tabulku je nutné označit atributem id s libovolnou hodnotou, která je po-té předávána jako parametr funkce dynamicTableInit. Script upravuje data v těle tabulky, tedy mezi tagy <tbody>. Části tabulky mezi tagy thead a tfoot ponechává skript beze změny. Funkčnost bya testována v prohlížečích Google Chrome, Internet Explorer, Edge a Mozilla Firefox.
Na konci HTML stránky je nutné zavolat inicializační funkci skriptu tímto způsobem:

	<script>
		dynamicTableInit("dynamicTable");
	</script>

Skript upravuje tabulku přidáním kontrolního řádnu jako prvního řádku do časti tabulky tbody. Velikost a písmo kontrolního řádku je možné upravovat pomocí css stylů. Více v kapitole Stylování níže.

Filtrování

Skript umožňuje filtrovat jednotlivé sloupce dle jejich obsahu. Pokud jsou data již podle nějakého sloupce filtrovány, je filtrování v dalších sloupcích omezeno pouze na již vyfiltrovaná data.

Řazení

Data v tabulce lze řadit podle některého ze sloupců, pomocí šipek sestupně nebo vzestupně v kontrolním řádku tabulky. U každého sloupce je možné definovat, zda se bude řadit jako řetězec nebo jako čísla. Tato definice se udává u každého sloupce v části thead tabulky. Pokud definice chybí, probíhá řazení sloupce jako řetězec. Řazení je možné aplikovat na tabulku i po-té, co byla data vyfiltrována.

Ukázka definice pro řazení sloupce jako řetězec a jako čísla:

  • <th data-sortSet="text">Company</th>
  • <th data-sortSet="number">Company</th>

Stylování

Kontrolní řádek tabulky, který je přidán skriptem, je možné vzledově upravit pomocí css souboru s názvem dynTable.css. Skript upravuje pouze část tabulky tbody, veškerý vzhled zbytku tabulky včetně vzhledu vstahující se na celou část tabulky tbody zůstává zachován. Ukázkový soubor se stylem je poskytován společně se skriptem, styl je možné upravovat pomocí těchto tříd:

  • dynTable_ControlCells - třída, která upravuje jednotlivé buňky kontrolního řádku
  • dynTable_ControlRow - třída upravuje vzhled celého kontrolního řádku, tento atribut je přídán k tagu tr
  • dynTable_inputCell - třída upravuje vzhled pole, pomocí kterého probíhá filtrování tabulky

Ukázka tabulky s použitím skriptu dynTable.js:

	<table id="dynamicTable">
		<thead>
			<tr>
				<th data-sortSet="text">Company</th>
				<th data-sortSet="text">Contact</th> 
				<th data-sortSet="text">Country</th> 
				<th data-sortSet="number">Number</th> 
			</tr> 
		</thead> 
		<tbody> 
			<tr> 
				<td>Alfreds Futterkiste</td> 
				<td>Maria Anders</td> 
				<td>Germany</td> 
				<td>10</td> 
			</tr> 
		</tbody> 
	</table>
